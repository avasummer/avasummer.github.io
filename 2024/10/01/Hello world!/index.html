Hello world!

在 Vidar-Team 的面试中，学长学姐问到了我有没有博客的问题。

在这之前，我确实没有一个自己的博客，也没有写博客的想法。不过在这之后，我认识到写博客确实是一种记录自己思考、学习经历的好方法。所以，不妨就来试试看吧，用文字留下自己的脚印，写下生活学习中的美好、彷徨、收获。

由于之前并没有搭建博客的经历，这对我也是一次接触新技术的机会。

本博客是利用 Hexo 和 Github Pages 运行的，可以说是比较方便、可靠的方法了。唯一的问题就是 Github 偶尔会出现无法访问的情况。

这里通过 Cloudflare 的 CDN 服务尽量避免这种情况的出现。

从写博客开始，我才开始接触怎么写 Markdown 、未来可能还会学习的 \(\LaTeX\) ，还有相关的网络知识之类的，要学的东西只会越来越多*^____^*。
